:wxfrwk:`物流助手-打单软件 <open-ability/authexpress-downloadorize>`
======================================================================

介绍
商户可以通过打单软件，自助打印面单

使用说明
通过 updatePrinter API 更新打印员。
打印员扫码登录、选择商户。
配置热敏打印机（PC端需安装打印机驱动，并选择203dpi分辨率）。
拉取面单，打印需要的面单；或启用自动打印，新订单下单成功即会自动打印。
最新版本下载地址
Windows

支持 Windows XP 及以上版本。

更新日志
