progress¶

1.0.0 新版功能: 低版本需做 兼容处理 。

组件属性的长度单位默认为px，2.4.0起支持传入单位(rpx/px)。

属性

类型

默认值

必填

说明

最低版本

number

否

百分比0~100

1.0.0

boolean

false

否

在进度条右侧显示百分比

1.0.0

border-radius

number/string

0

否

圆角大小

2.3.1

number/string

16

否

2.3.1

6

示例代码

在开发者工具中预览效果
