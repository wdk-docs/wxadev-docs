movable-view¶

1.2.0 新版功能: 低版本需做 兼容处理 。

可移动的视图容器，在页面中可以拖拽滑动。<movable-view>必须在 <movable-area> 组件中，并且必须是直接子节点，否则不能移动。

类型

默认值

必填

最低版本

none

否

movable-view的移动方向，

属性值有all、vertical、horizontal、none

1.2.0

false

否

1.2.0

out-of-bounds

boolean

false

否

超过可移动区域后， movable-view是否还可以移动

number

否

会自动移动到可移动范围；改变x的值会触发动画

number

否
