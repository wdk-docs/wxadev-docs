:wxapi:`wx.createLivePlayerContext`
============================================

LivePlayerContext

.. function:: wx.createLivePlayerContext(string id, Object this)


   .. versionadded:: 1.7.0 低版本需做 :ref:`compatibility` 。

创建 live-player 上下文 LivePlayerContext 对象。

参数
string id
<live-player> 组件的 id

Object this
在自定义组件下，当前组件实例的this，以操作组件内 <live-player> 组件

返回值
LivePlayerContext
