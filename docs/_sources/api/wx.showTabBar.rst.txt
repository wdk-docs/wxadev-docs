:wxapi:`wx.showTabBar`
============================================

.. function:: wx.showTabBar(Object object)

基础库 1.9.0 开始支持，低版本需做兼容处理。

显示 tabBar

参数
Object object
属性	类型	默认值	必填	说明
animation	boolean	false	否	是否需要动画效果
success	function		否	接口调用成功的回调函数
fail	function		否	接口调用失败的回调函数
complete	function		否	接口调用结束的回调函数（调用成功、失败都会执行）
