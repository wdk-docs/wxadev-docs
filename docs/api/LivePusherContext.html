LivePusherContext

class LivePusherContext()¶

LivePusherContext 实例，可通过 wx.createLivePusherContext() 获取。

livePusherContext 与页面内唯一的 <live-pusher> 组件绑定，操作对应的 <live-pusher> 组件。
