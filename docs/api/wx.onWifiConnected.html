wx.onWifiConnected¶

wx.onWifiConnected(function callback)¶

基础库 1.6.0 开始支持，低版本需做兼容处理。

监听连接上 Wi-Fi 的事件

参数 function callback 连接上 Wi-Fi 的事件的回调函数

参数 Object res 属性 类型 说明 wifi WifiInfo Wi-Fi 信息
