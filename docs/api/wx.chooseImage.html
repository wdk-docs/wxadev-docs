wx.chooseImage¶

wx.chooseImage({[count][ ,sizeType][ ,sourceType][ ,success][ ,fail][ ,complete]})¶
Label

从本地相册选择图片或使用相机拍照。

参数

  • count (number) – 9 最多可以选择的图片张数

  • sizeType (Array.<string>) –

    ['original', 'compressed'] 所选的图片的尺寸,合法值:

    • original 原图

    • compressed 压缩图

  • sourceType (Array.<string>) –

    ['album', 'camera'] 选择图片的来源,合法值:

    • album 从相册选图

    • camera 使用相机

  • success({tempFilePaths, tempFiles}) (function) –

    接口调用成功的回调函数

    • tempFilePaths (Array.<string>) 图片的本地临时文件路径列表

    • tempFiles (Array.<Object>) 图片的本地临时文件列表 1.2.0

      • path (string) - 本地临时文件路径

      • size (number) - 本地临时文件大小，单位 B

  • fail (function) – 接口调用失败的回调函数

  • complete (function) – 接口调用结束的回调函数（调用成功、失败都会执行）

示例
wx.chooseImage({
  count: 1,
  sizeType: ['original', 'compressed'],
  sourceType: ['album', 'camera'],
  success(res) {
    // tempFilePath可以作为img标签的src属性显示图片
    const tempFilePaths = res.tempFilePaths
  }
})
