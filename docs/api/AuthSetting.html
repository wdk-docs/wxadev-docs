AuthSetting¶

用户授权设置信息，详情参考权限

属性¶

scope.userInfo¶

是否授权用户信息

  • type - boolean

  • api - wx.getUserInfo()

scope.userLocation¶

是否授权地理位置，对应接口

  • type - boolean

  • api - wx.getLocation() , wx.chooseLocation()

scope.address¶

是否授权通讯地址

  • type - boolean

  • api - wx.chooseAddress()

scope.invoiceTitle¶

是否授权发票抬头

  • type - boolean

  • api - wx.chooseInvoiceTitle()

scope.invoice¶

是否授权获取发票，对应接口

  • type - boolean

  • api - wx.chooseInvoice()

scope.werun¶

是否授权微信运动步数，对应接口

  • type - boolean

  • api - wx.getWeRunData()

scope.record¶

是否授权录音功能，对应接口

  • type - boolean

  • api - wx.startRecord()

scope.writePhotosAlbum¶

是否授权保存到相册

  • type - boolean

  • api - wx.saveImageToPhotosAlbum() , wx.saveVideoToPhotosAlbum()

scope.camera¶

是否授权摄像头，对应<camera> 组件

  • type - boolean
