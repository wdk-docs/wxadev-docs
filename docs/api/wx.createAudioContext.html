wx.createAudioContext¶

wx.createAudioContext(string id, Object this)¶

1.6.0 版后已移除: 请使用 wx.createInnerAudioContext() 代替

Label

创建 audio 上下文 AudioContext() 对象。

参数

  • id (string) – <audio> 组件的 id

  • this (object) – 在自定义组件下，当前组件实例的this，以操作组件内 <audio> 组件

返回

AudioContext()
