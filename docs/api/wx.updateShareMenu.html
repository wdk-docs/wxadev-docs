wx.updateShareMenu¶

wx.updateShareMenu([withShareTicket][, isUpdatableMessage][, activityId][, templateInfo][, success][, fail][, complete])¶

1.2.0 新版功能: 低版本需做 兼容处理

更新转发属性

参数

  • withShareTicket (boolean) – false 是否使用带 shareTicket 的转发详情

  • isUpdatableMessage (boolean) –

    false 是否是动态消息，详见动态消息

    2.4.0 新版功能.

  • activityId (string) –

    动态消息的 activityId 。通过 updatableMessage.createActivityId() 接口获取

    2.4.0 新版功能.

  • templateInfo (object) –

    动态消息的模板信息

    2.4.0 新版功能.

    • parameterList (Array.<Object>) - 参数列表

      • name (string) - 参数名

      • value (string) - 参数值

  • success (function) – 接口调用成功的回调函数

  • fail (function) – 接口调用失败的回调函数

  • complete (function) – 接口调用结束的回调函数（调用成功、失败都会执行）

示例
wx.updateShareMenu({
  withShareTicket: true,
  success() { }
})
