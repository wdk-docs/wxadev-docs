updatableMessage.createActivityId¶

本接口应在服务器端调用，详细说明参见 服务端API。

本接口支持云调用。需开发者工具版本 >= 1.02.1904090（最新稳定版下载），wx-server-sdk >= 0.4.0

服务器端调用¶

GET message/wxopen/activityid/create?access_token=(ACCESS_TOKEN)&unionid=(UNIONID)¶
Synopsis

创建被分享动态消息或私密消息的 activity_id。详见 动态消息。

Parameters

  • cloudbase_access_token (string access_token /) – 是 接口调用凭证

  • unionid (string) – 否 为私密消息创建activity_id时，指定分享者为unionid用户。其余用户不能用此activity_id分享私密消息。openid与unionid填一个即可。私密消息暂不支持云函数生成activity id。

  • openid (string) – 否 为私密消息创建activity_id时，指定分享者为openid用户。其余用户不能用此activity_id分享私密消息。openid与unionid填一个即可。私密消息暂不支持云函数生成activity id。

Request JSON Object

  • activity_id (string) – 动态消息的 ID

  • expiration_time (number) – activity_id 的过期时间戳。默认24小时后过期。

  • errcode (number) – 错误码 0 请求成功 -1 系统繁忙。此时请开发者稍候再试 42001 access_token 过期
