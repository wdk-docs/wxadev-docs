wx.qy.getNFCReaderState¶

wx.qy.getNFCReaderState({[success][, fail][, complete]})¶
Label

判断当前设备是否支持 NFC 能力

参数

  • success({errcode,errmsg}) (function) –

    接口调用成功的回调函数

    • errcode (number) - 错误码

      • 0 正常

      • 13000 当前设备不支持NFC

    • errmsg (string) - 错误信息

      • ok 正常

  • fail (function) – 接口调用失败的回调函数

  • complete (function) – 接口调用结束的回调函数（调用成功、失败都会执行）

前提

需要用户授权

示例
wx.qy.getNFCReaderState({
  success: (res) => {
    console.log(JSON.stringify(res))
    // todo what you want
  },
  fail: (res) => {
    wx.showModal({
      title: 'NFC-Not-Support',
      content: JSON.stringify(res),
    })
  }
})
