wx.qy.getAvatar¶

wx.qy.getAvatar({[timeout][, success][, fail][, complete]})¶
Label

获取企业成员头像

Parameters

  • timeout (Number) – 否 超时时间，单位 ms

  • success({avatar}) (function) –

    接口调用成功的回调函数

    • avatar (String) - 当前企业成员的头像url

  • fail (function) – 接口调用失败的回调函数

  • complete (function) – 接口调用结束的回调函数（调用成功、失败都会执行）

前提

  1. 必须先调用过 wx.qy.login()，且session_key未过期，开发者可调用 checkSession 检查当前登录态

  2. 要求用户在应用可见范围内，且需要用户同意确认

示例代码
wx.qy.getAvatar ({
  success: function(res) {
    var avatar = res.avatar
  },
  fail: function(res) {
    console.log(res.fail_reason)
  }
})
