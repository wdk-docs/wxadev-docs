数据分析接口¶

开发者通过数据分析接口，可获取到小程序的各项数据指标，便于进行数据存储和整理。数据分析详细功能介绍及指标解释参见数据分析文档。

相关接口 概况 访问趋势 日趋势 周趋势 月趋势 访问分布 访问留存 日留存 周留存 月留存 访问页面 用户画像 自定义数据上报
