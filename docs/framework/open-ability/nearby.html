附近的小程序¶

概述 附近的小程序 API，提供给需快速批量创建附近地点的小程序开发者使用，使用前请先调高附近地点额度。 调高附近地点额度申请方式如下：

下载 《调高地点额度申请表》，填写后发送至 placeofminiprogram@qq.com。

邮件主题为：“帐号名称”申请调高附近地点额度。审核通过后可调高额度。

管理接口 添加地点 查看地点列表 删除地点 展示/取消展示附近小程序
